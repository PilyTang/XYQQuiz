# 安全策略

## 支持范围

当前仅维护最新的 `0.1.x` 版本。请先在最新版复现问题，并保留版本报告和最小必要的环境信息。

## 报告安全问题

请使用 GitHub 仓库 Security 页面中的私密安全报告功能，不要在公开 Issue 中披露可利用细节、令牌、角色名、聊天内容或完整诊断包。

如果仓库暂未启用私密报告，请先只提交不含敏感细节的公开问题，说明需要私下联系维护者；不要直接上传“保存识别诊断”生成的 ZIP。

## 诊断数据

识别诊断可能包含完整游戏画面、OCR 裁剪和日志尾部。程序会对已知令牌、凭据字段和本机路径做基础脱敏，但无法保证自动识别画面里的个人信息。分享前请人工检查。

项目没有遥测，不会自动上传画面、题目、日志或诊断文件。Web 服务只允许绑定 `127.0.0.1`。
