# Changelog

本项目遵循语义化版本号。

## [0.1.0] - 2026-07-14

首个公开候选版本：

- Windows 10/11 x64 便携式 `onedir` 应用和本地浏览器界面。
- Windows Graphics Capture 游戏窗口预览。
- 离线科举题库、题库原子更新和高置信度答案框。
- 普通题、图片题、3～4 选项、多分辨率和可变题面高度识别。
- 会试、殿试布局适配；乡试兼容实现待活动现场验证。
- 单实例激活、一次性浏览器令牌、会话恢复和端口冲突提示。
- 识别诊断、无画面环境诊断、一键自检和构建清单校验。
- GitHub CI 与标签发布流程。
