XYQQuiz 梦幻西游科举答题助手
==============================

支持范围：Windows 10 1903 及以上、Windows 11，仅限 x64。
Windows 11 x64 已验证；Windows 10 1903+ 为目标兼容范围，首版尚未实机验证。

使用方法：
1. 完整解压 ZIP，不要直接在压缩包内运行。
2. 双击 XYQQuiz.exe。
3. 首次正常启动会请求 UAC 管理员权限；允许后等待浏览器自动打开。
4. 第二次双击只会重新打开已运行页面，不会再次请求 UAC。
5. 网页中的“退出程序”会停止捕获并关闭后台。

离线能力：发布包自带题库、布局、锚点和 OCR 模型，首次启动不需要联网。
网页“更新题库”只更新题库，不会更新程序。

识别范围：支持普通文字题、图片题、3～4 个选项和多种窗口分辨率。
会试、殿试已有实际界面适配；乡试尚未完成活动现场验证。

自检：双击“一键自检.cmd”。报告会写入 diagnostics\self-test-latest。

升级：把新版解压到全新目录，再复制旧目录中的 config.json、data、logs、diagnostics。
不要把新版 EXE 或 _internal 覆盖到旧版目录。

安全提示：首版未购买代码签名证书，SmartScreen 可能显示未知发布者。
请从项目 GitHub Release 下载并核对 SHA-256，不要全局关闭杀毒软件。

诊断隐私：“保存识别诊断”会写入当前完整游戏画面、OCR 裁剪和日志尾部。
分享诊断 ZIP 前请人工检查角色名、聊天和其他个人信息；程序不会自动上传。

本项目为非官方辅助显示工具，不注入游戏、不读取进程内存、不自动点击答案。
